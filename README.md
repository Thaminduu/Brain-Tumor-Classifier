# Brain-Tumor-Classifier

A deep learning-based image classification project that classifies brain MRI images into tumor-related categories using a convolutional neural network.

The project explores the use of computer vision in medical image analysis by applying preprocessing, model training, validation, and evaluation techniques on a brain tumor MRI dataset. The goal was to understand how CNN-based models can be used for image classification tasks in healthcare-related applications.

> This project is for educational and research purposes only. It is not intended for clinical diagnosis or medical decision-making.
